# Mekanismplus
어떤유저를 위한 모드?
- 버전:1.19.2
- 염화수소,황산 기체,핵연료를 수억 mb이상 대량생산 및 단순하고 효율적인 공정을 원하는 경우.
- 바닐라에서는 우라늄 및 기타 광석그리고 다이아몬드가 충당이 되지 않는다고 생각하는 경우.
- qio에서 주괴 수백만개를 모아서 코스트코를 마인크래프트에서 실현하고 싶은 경우.
- 오버월드 와 공장 전용 차원 이원화를 원하는 경우.
